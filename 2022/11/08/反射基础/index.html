反射

获取类

一切的操作都需要先获取类，三种方法

  • Class c = Class.forName(类的全限名)
  • Class c = 类.class
  • Class c = 类.getClass()

获取构造器

  • getConstructors() 获取所有***public*** 的构造器
  • getDeclareConstructors() 获取所有构造器
  • getConstructor() 获取单个***public*** 的构造器，由参数决定是哪个
  • getDeclareConstructors() 获取单个构造器，由参数决定是哪个，参数是类型加***.class***
  • 得到构造器对象之后，可以newInstance() 创建对象
  • 如果是非public的构造器，setAccessible(true)可以破坏封装性

获取成员变量

  • getDeclaredFields() 获取所有成员变量
  • getDeclaredField(String name) 获取单个变量
  • 得到成员变量之后可以取赋值
  • set(对象，值)
  • get(对象)
  • setAccessible(true)

获取方法

  • getDeclaredMethods() 获取所有方法
  • getDeclaredMethod(name,参数（没有就不用写) ) 获取单个方法
  • 得到方法对象后可以执行
  • object invoke(对象，参数（没有就不用写) ) void类型就返回NULL

泛型擦除

泛型只在编译阶段有约束 就像厕所类型一样，在运行阶段都是ArrayList类

不然会产生很多类，如：ArrayList ArrayList

    t.add(1);
    ArrayList T = t;
